## ✨ 目前功能

> 📌 **所有指令前面都要加 `$$` + 空白**  
> 📌 **指令不分大小寫**

### 🎵 音樂播放相關

| 功能 | 指令 |
|---|---|
| 播放音樂（需 YouTube 網址） | `play` / `p` `<網址>` |
| 跳過目前歌曲 | `skip` / `s` |
| 查詢音樂（播放搜尋結果第一首） | `find` / `f` `<關鍵字>` |
| 播放清單列表 | `list` / `li` |

---

### 🔁 播放控制

| 功能 | 指令 |
|---|---|
| 循環播放 | `loop` / `lo` |
| 取消循環 | `unloop` / `u` |

---

### 🤖 自動推薦

| 功能 | 指令 |
|---|---|
| 自動推薦歌曲（< 10 分鐘） | `relate` / `r` |

- 再輸入一次可取消推薦

---

### 🔊 特殊功能（⚠️ 危險）

| 功能 | 指令 |
|---|---|
| 瘋狂爆音模式 | `e` / `爆` |

⚠️ 再次輸入可恢復正常  
⚠️ 恢復後歌曲可能會有一點卡卡（屬正常現象）
